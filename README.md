# Assignment_0

## Overview
This assignment asks students to implement a function in Python and check it with unit tests.

- `assignment.py`: Implement the function `sum_of_3(a, b, c)`.
- `unit_tests.py`: Run these tests to verify your implementation.

## Instructions

1. Implement `sum_of_3(a, b, c)` in `assignment.py`.
